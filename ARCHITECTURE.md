## Data Query Flow (Dashboard)

```
User Visits /mission-control/login
         в”Ӯ
         в”ңв”Җв”Җ NextAuth checks session
         в”Ӯ
         в”ңв”Җ If not authenticated
         в”Ӯ  в””в”Җ Redirect to /mission-control/login
         в”Ӯ
         в””в”Җ If authenticated
            в”Ӯ
            в””в”Җ Load mission-control/page.tsx
               в”Ӯ
               в””в”Җ Call getDashboardMetrics()
                  в”Ӯ
                  в”ңв”Җ Promise.all([
                  в”Ӯ   в”ңв”Җ COUNT(*) FROM visitors
                  в”Ӯ   в”ңв”Җ COUNT(*) FROM visitors WHERE created_at >= today
                  в”Ӯ   в”ңв”Җ COUNT(*) FROM visitors WHERE created_at >= 7 days ago
                  в”Ӯ   в”ңв”Җ COUNT(*) FROM visitors WHERE created_at >= 30 days ago
                  в”Ӯ   в”ңв”Җ COUNT(*) FROM contact_messages
                  в”Ӯ   в”ңв”Җ COUNT(*) FROM contact_messages WHERE is_read = false
                  в”Ӯ   в”ңв”Җ COUNT(*) FROM assistant_interactions
                  в”Ӯ   в”ңв”Җ SELECT * FROM contact_messages ORDER BY created_at DESC LIMIT 1
                  в”Ӯ   в”ңв”Җ SELECT * FROM assistant_interactions ORDER BY created_at DESC LIMIT 20
                  в”Ӯ   в”ңв”Җ SELECT * FROM visitors ORDER BY created_at DESC LIMIT 20
                  в”Ӯ   в””в”Җ SELECT * FROM visitors (all, for aggregation)
                  в”Ӯ  ])
                  в”Ӯ
                  в”ңв”Җ Client-side aggregation:
                  в”Ӯ  в”ңв”Җ Count unique countries (Set)
                  в”Ӯ  в”ңв”Җ Group by country (Map)
                  в”Ӯ  в”ңв”Җ Sort countries by count DESC
                  в”Ӯ  в”ңв”Җ Group by city (Map)
                  в”Ӯ  в””в”Җ Sort cities by count DESC
                  в”Ӯ
                  в””в”Җ Return DashboardMetrics
                     в”Ӯ
                     в””в”Җ Pass to <AdminDashboard metrics={...} />
                        в”Ӯ
                        в””в”Җ Render all 8 visual panels with live data
```

## Visitor Tracking Flow

```
User navigates to homepage
         в”Ӯ
         в””в”Җ React component mounts
            в”Ӯ
            в””в”Җ <VisitorTracker /> component loads
               в”Ӯ
               в””в”Җ useEffect hook runs
                  в”Ӯ
                  в”ңв”Җ fetch ipapi.co/json/
                  в”Ӯ  в””в”Җ Get { country_name, city }
                  в”Ӯ
                  в”ңв”Җ Hash IP: SHA256(IP + "mission-log-salt")
                  в”Ӯ  в””в”Җ Generate 64-char hex string
                  в”Ӯ
                  в”ңв”Җ Check for duplicate
                  в”Ӯ  в””в”Җ SELECT * FROM visitors
                  в”Ӯ     WHERE ip_hash = hash
                  в”Ӯ     AND created_at >= (now - 5 minutes)
                  в”Ӯ
                  в”ңв”Җ If found (duplicate within 5 min)
                  в”Ӯ  в””в”Җ Return { ok: true, isDuplicate: true }
                  в”Ӯ
                  в””в”Җ If not found
                     в”ңв”Җ INSERT { country, city, ip_hash, user_agent, created_at }
                     в””в”Җ Return { ok: true, isDuplicate: false }

Non-blocking request doesn't affect page load
                     в”Ӯ
                     в””в”Җ Visitor appears in dashboard
                        within seconds
```
